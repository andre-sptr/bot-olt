Peran OpenAI
AI Config Assistant

Mengubah instruksi bahasa Indonesia menjadi draft konfigurasi terstruktur. User tetap melakukan preview, test, dan aktivasi.

AI Failure Analyst

Meringkas log gagal dan memberi langkah perbaikan. AI tidak memiliki izin menjalankan shell atau kode bebas.

Boundary keamanan

API key WAHA dan OpenAI disimpan terenkripsi. Worker hanya menjalankan operasi yang telah ditentukan. Tidak ada Python hasil AI yang dieksekusi langsung.
